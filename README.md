# Matrix Operations using Signals, Pipes & FIFOs

## Overview

This project implements a Linux-based matrix operations system using multiprocessing techniques and Interprocess Communication (IPC).

The application allows users to create, manage, and process matrices through an interactive menu-driven interface. Matrix operations are distributed among child processes using Linux IPC mechanisms such as Signals, Pipes, and FIFOs.

## Features

* Matrix Creation and Management
* Matrix File Loading and Saving
* Matrix Addition
* Matrix Subtraction
* Matrix Multiplication
* Determinant Calculation
* Eigenvalue and Eigenvector Computation
* Multiprocessing Architecture
* Process Pool Management
* OpenMP Performance Optimization

## Technologies Used

* C Programming
* Linux System Programming
* Signals
* Pipes
* FIFOs
* OpenMP
* Process Management
